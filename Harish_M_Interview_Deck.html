06 / Project 02 · NIT Silchar · Published

PneumoScan,
X-ray AI that
knows its doubt

A Vision Transformer classifies chest X-rays as normal or pneumonia, and reports uncertainty through Monte Carlo dropout, twenty stochastic passes turned into predictive entropy. Active learning reached strong accuracy on a fraction of the labels.

The headline, about 87% accuracy on just 250 labels across three rounds, against a 72% CNN baseline. A fifteen point gain on roughly 75% fewer labels. Deployed live on Azure App Service.

07 / Project 03 · Data & LLM systems

Samudhraa,
talk to the ocean

A retrieval augmented chatbot over Argo style oceanographic data. Ask in plain English, it grounds the answer on Chroma vector retrieval and a guarded SQL select over PostgreSQL, then renders a chart when the query is analytical.

Chat and embeddings run on Azure AI Foundry with no local weights. Postgres and Chroma run as containers. The whole stack starts on one Docker Compose command. Semantic search accuracy above 85%.
